# devres.co

DevRes.co is a website that collected awesome developer's resources!

https://devres.co/

## Contribution welcome!

## How to contribute

1. Copy TEMPLATE.yml to (service name).yml. eg. ec2.yml
2. Write content
3. Send PR
4. Done!

## What is developer resources?

- The web service has API
- IoT devices like Raspberry Pi, Arduino and etc.
- The web app for developer and developer team like project management
